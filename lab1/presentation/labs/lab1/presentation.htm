Методы кодирования и модуляции сигналов

Студент: Зашихалиева З.Р.

Дата: 29.03.2026

Цель работы

Изучение методов кодирования и модуляции сигналов с помощью Octave.

Задачи

  1. Построить графики функций
  2. Разложить меандр в ряд Фурье
  3. Найти спектр сигналов
  4. Сделать амплитудную модуляцию
  5. Реализовать 6 методов кодирования

Установка Octave

  • Установка: sudo apt install octave
  • Версия: 8.4.0

Построение графиков функций

Формулы: - y1 = sin x + (1/3) sin 3x + (1/5) sin 5x - y2 = cos x + (1/3) cos 3x + (1/5) cos 5x

Разложение меандра

  • Меандр содержит только нечетные гармоники
  • Чем больше гармоник, тем точнее форма сигнала

Спектральный анализ

Параметры сигналов: - Сигнал 1: 10 Гц, амплитуда 1 - Сигнал 2: 40 Гц, амплитуда 0.7

Амплитудная модуляция

  • Низкочастотный сигнал: 5 Гц
  • Несущая частота: 50 Гц
